## Краткая справка
### Функции

Объявление задает имя функции, тип возвращаемого значения и типы аргументов

```cpp
Elem∗ next_elem(); // нет аргументов, возвращает Elem*

void exit(int); // один аргумент типа int, ничего не возвращает

double sqrt(double); // аргумент типа double, возвращает double
```
Функция должна быть объявлена преждем чем может быть использована (сколько угодно раз)
Функция должна быть определена преждем чем может быть использована (только 1 раз с точностью до токенов, поскольку ODR, иначе UB). 

**Определение** (функции) - объявление в котором также определено тело функции

```cpp
void swap(int∗, int∗); // объявление
void swap(int∗ p, int∗ q) // определение
{
    int t = ∗p;
    ∗p = ∗q;
    ∗q = t;
}
```

Аргументы функций - копии переданных значений. Изменение аргумента внутри тела функции не меняет оригинальное значние переданное в момент вызова функции.

Параметры по умолчанию могут идти только в конце. Параметры по умолчанию задаются только в объявлении, поскольку вызывающая функция видит только определение.

Параметры по умолчанию нельзя повторять или менять в объявлениях в пределах одной облаcти видмости

Можно делать перегрузки. Формальные правила разрешения перегрузок довольно душные: https://en.cppreference.com/cpp/language/overload_resolution. Тип возвращаемого значение в перегрузке не участвует.

### Задание 2
Реализуйте функцию max которая возвращает максимум из 2 чисел. Аргументы функции и возвращаемое значение должны иметь тип `int`. Решение должно содержать определение 1 функции в файле `solution.cpp`.

### Задание 3

### Задание 4
Реализуйте функцию `bool check_password(std::string, int min_length)` которая проверяет пароль на соответствие условиям:

1 - Минимум 1 заглавная буква

2 - Минимум 2 строчная буква

3 - Минимум 1 цифра

4 - Минимум 1 спецсимвол из списка: `!#$%^*`

5 - Строка содержит не менее `min_length` символов, причём `min_length` должен быть аргументом со значением по умолчанию `8`

### Задание 5
Реализуйте функцию `int find_char(std::string, int from, int to)` которая ищет символ в строке:

- Аргументы `from` и `to` задают полуинтервал поиска (от `from` включительно, до `to` не включительно).
- Аргументы `from` и `to` должны быть аргументами по умолчанию. Если аргумент `from` не указан, то поиск ведётся с начала строки. Если аргумент `to` не указан, то поиск ведётся до конца строки. Заметьте, что указать `to` не указав `from` невозможно.
- Если символ не найдет, функция должна вернуть `-1`, иначе индекс первого вхождения символа в строку.
